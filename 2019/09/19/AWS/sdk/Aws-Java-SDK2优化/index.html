    HTTP异步配置

    基于netty的异步非阻塞io

    最大连接数：

    可以通过使用 maxConcurrency 方法来设置允许打开的最大 HTTP 连接数。您可以使用 maxPendingConnectionAcquires 方法设置在达到最大并发数后，允许排队的最大请求数。

    maxConcurrency 默认值：50

maxPendingConnectionAcquires　默认值：10_000

    将最大连接数设置为并发事务的数量可避免连接争用和性能不佳。
